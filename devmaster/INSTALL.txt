DevShop Hosting
===============

devshop_hosting is the front-end code for DevShop.  This module requires the
devshop distribution to work.

Documentation is located in the main project: http://drupal.org/project/devshop

Installation
------------
Don't install this module by itself. It needs the entire devshop system to
function.

For installation instructions, see http://drupal.org/project/devshop
