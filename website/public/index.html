The Open Source DevOps Framework

DevShop is an Open Source DevOps Framework and a set of reusable components designed to improve the web development, testing, and server management process, especially for Drupal websites.

The goal of DevShop is to provide a complete web development & testing pipeline out of the box, while empowering users to choose their own server infrastructure and systems architecture.

DevShop uses Ansible & Ansible Galaxy for server configuration, Symfony Console for command line tools, and Drupal for the web interface.

A complete Drupal CI/CD & QA/QC solution.

  • DevShop provides an all-in-one solution for Continuous Integration and Delivery for Drupal.
  • Launch unlimited environments on a single server to keep costs low during development and testing.
  • Built-in continuous integration launches new environments automatically when developers need them. No scripts or third-party integrations to maintain.

A Server Management System, powered by Ansible, built on Drupal.

  • DevShop is a Server Management System that lets you track your server inventory and configure them from a central web interface.
  • Set up your servers with the included Ansible Roles or add your own, using Drupal as the Ansible Inventory for your own cloud.
  • Edit a server’s configuration as easily as you edit your website’s content, in a simple web interface using basic Yaml syntax.
  • Launch, configure and destroy new servers using the DevShop web interface and your favorite cloud provider.

Run anywhere with a 100% Open Source stack.

  • DevShop is 100% open source, allowing it to be run in any data center using Linux.
  • Choose your own server provider and network architecture to maximize functionality, minimize costs, and comply with the contraints of your organization.
  • DevShop provides open source server configuration at the operating system and services level, making it compatible with any configuration management, security, or support system such as RHEL. Expand to containers and orchestrators at your own pace.
